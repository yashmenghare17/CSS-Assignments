Positions

Position are as follow :

  • Fixed
  • Static
  • Relative
  • Sticky
  • Absolute

Fixed

Element position will be relative to its viewport[visible space in browser]


Static

Default position value and there is no effect of top,bottom,left and right respectivly

Every element without position have default postion i.e static


Relative

Element position will be relative to its orignal position


Sticky

Element position will be relative if it is in viewport else it will act as fixed
